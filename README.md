# 私人订制Cli

#### 个人写着玩的,目前私人订制化较高，只支持react+typescript+redux，之后可能继续拓展

### 安装步骤
+ git clone
+ cd zhang-cli
+ npm install
+ npm link

#### 安装完成后，可以使用 zhang --help 来查看是否安装成功

### 使用文档
+ 创建项目 
+    zhang create projectName
+ 添加Page 
+    zhang addPage pageName
+ 添加Component
+    zhang addComponent componentName
